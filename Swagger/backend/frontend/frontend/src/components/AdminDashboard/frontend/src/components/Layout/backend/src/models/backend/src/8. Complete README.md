# 🎉 Voting System MVP with Paystack Integration

A real-time online voting platform with integrated payment functionality using Paystack, built with modern full-stack technologies.

## ✨ Features

### User Features
- ✅ Real-time voting dashboard with live vote counts
- ✅ Secure payment integration via Paystack
- ✅ Duplicate vote prevention
- ✅ Payment verification before voting

### Admin Features
- ✅ Candidate management (CRUD)
- ✅ Real-time vote monitoring
- ✅ Payment transaction tracking
- ✅ Results export (CSV/Excel)
- ✅ Analytics dashboard with charts
- ✅ Multi-election support

### Technical Features
- ✅ WebSocket real-time updates
- ✅ JWT authentication
- ✅ Rate limiting & security
- ✅ Responsive UI (Mobile-friendly)
- ✅ Docker containerization
- ✅ Production-ready deployment
